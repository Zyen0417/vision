# 视力分级预测计算器

基于有序 Logistic 回归的视力概率预测工具，用于临床科研参考。

## 功能说明

支持四种检查距离的视力等级预测：

| 距离 | 预测等级 |
|------|---------|
| 30cm | 较差 / 中等 / 较好（三级） |
| 40cm | 较差 / 中等 / 较好（三级） |
| 60cm | 中等 / 较好（二级） |
| 80cm | 中等 / 较好（二级） |

## 使用方法

1. 选择检查距离（30/40/60/80 cm）
2. 输入对应的临床参数：
   - **30cm**：眼轴、WTW
   - **40cm**：眼轴、WTW
   - **60cm**：年龄、眼轴、瞳孔直径、Km
   - **80cm**：轴率比（眼轴、Km）
   * 轴率比根据轴率比=眼轴*Km/337.5自动计算
3. 点击「计算所有距离的视力概率」查看结果

## 模型来源

系数基于临床统计 Logistic 回归模型，请勿替代医生诊断。
